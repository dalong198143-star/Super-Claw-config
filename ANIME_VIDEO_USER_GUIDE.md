## 📖 使用步骤

### 第一步：选择工具

1. 在左侧边栏找到 **🎬 动漫视频** 工具卡片
2. 点击进入动漫视频创作界面

### 第二步：创意构思（步骤1/4）

#### 输入提示词
在文本框中详细描述你想要的动漫场景，例如：
```
一个穿着粉色裙子的魔法少女站在樱花树下，手中拿着魔法杖，周围飘着粉色的花瓣，阳光透过树叶洒下斑驳的光影
```

#### 负面提示词（可选）
输入不希望出现的内容：
```
low quality, blurry, deformed hands, extra limbs
```

#### 选择动漫风格
从6种风格中选择：
- 🎨 **吉卜力风** - 宫崎骏风格，温暖治愈
- ✨ **现代日漫** - 精致线条，鲜艳色彩
- 🌸 **复古动漫** - 90年代经典风格
- 🎭 **Q版萌系** - 可爱大头娃娃
- 🌃 **赛博朋克** - 未来科技风
- 🏰 **奇幻冒险** - 魔法世界风格

点击"下一步：配置参数 →"继续

---

### 第三步：参数配置（步骤2/4）

#### 视频时长
- 3秒 - 短视频片段
- 5秒 - 标准时长 ⭐推荐
- 8秒 - 完整场景
- 10秒 - 长镜头

#### 分辨率
- 720P (1280×720) - 高清
- 1080P (1920×1080) - 全高清 ⭐推荐

#### 帧率 (FPS)
- 12 FPS - 传统动画感
- 24 FPS - 电影标准 ⭐推荐
- 30 FPS - 流畅播放

#### 运动强度
- 🌊 轻微 - 柔和缓慢
- 💫 中等 - 自然流畅 ⭐推荐
- 🔥 强烈 - 动感十足

#### 镜头运动
- 📷 固定镜头 - 无摄像机移动
- 🔍 推近 - 缓慢推进
- 🔭 拉远 - 逐渐远离
- ⬅️ 左移 - 水平左移
- ➡️ 右移 - 水平右移
- 🔄 环绕 - 环绕主体

#### 运动模板（快速填充）
点击任意模板自动填充运动描述：
- 🌊 微风吹拂
- 📹 镜头推近
- 🌸 花瓣飘落
- 🎆 烟火绽放
- 💃 人物舞动
- ☁️ 云层流动
- 🔥 火焰跃动
- 💧 水波涟漪

#### 自定义运动描述（可选）
如果需要更精确的控制，可以手动输入运动描述

#### 种子值（可选）
输入数字以复现相同结果，留空则随机生成

点击"下一步：生成图片 →"继续

---

### 第四步：生成首帧（步骤3/4）

1. 查看即将生成的首帧信息摘要
2. 点击"🎨 生成首帧图片"按钮
3. 等待AI绘制首帧（进度条显示）
4. 预览生成的首帧图片
5. 如果不满意，可以点击"🔄 重新生成"
6. 满意后点击"🎬 生成视频 →"继续

---

### 第五步：生成视频（步骤4/4）

1. 查看视频生成参数摘要
2. 点击"🚀 开始生成视频"按钮
3. 观察4阶段进度：
   - ✓ 分析首帧 (20%)
   - ✓ 生成关键帧 (50%)
   - ✓ 插值补帧 (80%)
   - ✓ 渲染输出 (100%)
4. 等待视频生成完成
5. 预览生成的视频
6. 点击"📥 下载视频"保存到本地
7. 或点击"🔄 重新创作"开始新的视频

### 常见问题

**Q: 为什么生成的视频不够流畅？**
A: 尝试提高帧率到30 FPS，或降低运动强度

**Q: 如何获得更好的画面质量？**
A: 使用1080P分辨率，添加详细的提示词描述

**Q: 视频生成失败怎么办？**
A: 检查网络连接，简化提示词，或尝试不同的参数组合

**Q: 如何复现喜欢的效果？**
A: 记录使用的种子值，下次输入相同的种子值即可

## 🎯 进阶技巧

### 1. 提示词优化公式
```
[主体描述] + [环境背景] + [动作状态] + [光影效果] + [风格修饰]
```

### 2. 运动描述技巧
- 使用动词描述动态：falling, flowing, dancing, rotating
- 指定运动方向：left to right, upward, clockwise
- 描述运动速度：slowly, gracefully, rapidly

### 3. 风格混合
可以在提示词中混合多种风格元素，创造独特效果

### 4. 批量测试
使用相同的提示词，改变种子值生成多个变体，选择最佳效果
